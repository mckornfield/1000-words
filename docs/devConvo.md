# Developer Chat

A structured log of development discussions and decisions.

## Format

```
<User> | [TAG-###]/DD-MON-YY/HH:MM ~> Title/Subject: <Message>
```

## Example

```
Alex | [FEAT-001]/10-JUN-26/18:00 ~> New Dashboard: Implemented responsive layout for mobile devices
```


## Tags

- `[GEN-###]` - General discussion
- `[BUG-###]` - Bug report or fix
- `[FEAT-###]` - Feature request or implementation
- `[QSTN-###]` - Question or clarification
- `[DECSN-###]` - Architecture or design decision
- `[TODO-###]` - Task or action item
